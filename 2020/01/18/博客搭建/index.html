博客搭建

1.为什么使用hexo+GitHub的方式建设个人博客

通俗的讲就是穷，买不起服务器。hexo+GitHub的方式可以在不使用服务器的情况下可以免费建站。

2. 博客搭建的准备工作

2.1 准备git，node.js的安装，GitHub账号

Git下载 || node.js下载 || GitHub注册

3.开始我们的正题

3.1 下载hexo

顺便创建一个博客使用的本地文件夹，在文件夹里面右键选择 git bash 。
安装和开始制作blog

npm install hexo-cli -g 
hexo init

启动本地博客

hexo g`

hexo s

点击 localhost:4000 有内容就是成功了。

3.2 下载hexo的主题

比如下载concise主题

git clone <https://github.com/HmyBmny/hexo-theme-concise.git> themes/concise `

修改_config.yml中的theme 冒号后面空格

`theme: concise`

4.使用hexo与GitHub连接

4.1创建github项目

4.2配置ssh_key密钥

要使用 git 工具首先要配置一下SSH key，为部署本地博客到 Github 做准备。

打开命令行输入 cd ~/.ssh 如果没报错或者提示什么的说明就是以前生成过的，直接使用 cat ~/.ssh/id_rsa.pub 命令就是可以查看本机上的 SSH key 了。

cat ~/.ssh/id_rsa.pub

如果之前没有创建过使用下面命令创建

git config --global user.name "用户名"
git config --global user.email "邮箱地址"

开始生成密钥

ssh-keygen -t rsa -C '上面的邮箱'`

首次使用还需要确认并添加主机到本机SSH可信列表。若返回 Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access. 内容，则证明添加成功。

ssh -T git@github.com `

登录GitHub，找到settings。有个 SSH and GPG keys标题。创建新的密钥。名字随便起，内容CTRL+C即可。
